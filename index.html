My Projects

The Snake Game(Kepler-452B Terminator)

Technology Used: Javascript, HTML, CSS

Description: The Kepler-452B Terminator is an engaging web game controlled by the arrow keys (Up, Down, Left, Right). The game starts with a snake of two blocks in a 500x600 square arena. The player directs the snake to eat randomly appearing apples using the arrow keys. Each apple eaten makes the snake one block longer. The game is won when the snake fills the entire arena, and it's lost if the snake's head hits a wall, triggering a reset. The game begins when the user presses any arrow key, starting the apple-spawning process.

Habo-Berlin Management System

Technology Used: Javascript, HTML, CSS, Express, MongoDB

Description: The Habo-Berlin Guardian Registration System provides secure CRUD operations for a cutting-edge data center in Berlin. Hosting 36 advanced Guardians—Atlas, Norma, and Clasper—it works with police units to tackle serious crimes like armed robbery, hostage rescues, and terrorism. Due to the high-risk nature of their duties, Guardians often sustain damage, requiring up-to-date records for repairs and replacements. Habo Industry, the manufacturer, continuously innovates, making the data center's secure CRUD functionality essential for ensuring the Guardians are equipped with the latest technologies.
